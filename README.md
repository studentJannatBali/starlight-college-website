## Starlight College Website

A responsive, multi-page college website developed as a frontend academic project using **HTML5, custom CSS, and Bootstrap**.  
The project focuses on clean UI design, structured layouts, and mobile-first responsiveness.

This website simulates a real-world college portal, covering essential sections such as admissions, academics, faculty, and contact information.

---

##  Project Objectives
- Design a professional and visually appealing college website
- Apply real-world HTML & CSS layout techniques
- Implement responsive design using Bootstrap
- Improve frontend structuring and navigation skills

---

##  Key Features
- Fully responsive design (mobile, tablet, desktop)
- Clean and modern user interface
- Multi-page structure:
  - Home
  - Admissions
  - Academics
  - Faculty
  - About
  - Contact
- Bootstrap components for layout and responsiveness
- Well-structured and readable codebase
- Easy navigation between pages

---

##  Tech Stack
- **HTML5** – Semantic page structure  
- **CSS3** – Custom styling and layouts  
- **Bootstrap** – Responsive grid system and components
