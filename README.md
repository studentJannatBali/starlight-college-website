# 🌟 Starlight College Website 

A modern, interactive college website featuring dynamic event management, client-side data persistence, search functionality, and an advanced admin dashboard. Built as a frontend project to demonstrate real-world JavaScript skills.

---

## 📋 Overview

This project simulates a complete college portal with interactive forms, event registration, and admin tools. It uses LocalStorage and JSON-based dynamic data loading, making it scalable and realistic.

🔗 Live Demo:  
https://studentjannatbali.github.io/starlight-college-website/

### 📝 Interactive Forms
- Contact form stores messages with timestamps  
- Admission form saves student applications  
- Real-time validation for inputs  
- Instant feedback after submission  

---

### 🎯 Event Management System
- Dynamic events loaded from `events.json`  
- Event registration system (LocalStorage)  
- Live search functionality for events  
- Dynamic event cards using JavaScript  

---

### 🔔 Toast Notifications
- Modern UI notifications instead of alerts  
- Auto-dismiss after 3 seconds  
- Used for user actions like event registration  

---

### 📊 Admin Dashboard (Hidden Feature)
- Double-click footer to open  
- View all stored data in console  
- Export data as JSON file  
- Clear all stored data  
- Real-time statistics display  

---

### 💾 Data Persistence
- Uses LocalStorage API  
- Stores:
  - Contact messages  
  - Admission applications  
  - Event registrations  
- Each entry includes ID and timestamp  

---

### 🎨 User Experience
- Responsive design (mobile + desktop)  
- Clean UI with Bootstrap  
- Smooth interactions  
- Lightweight (no backend required)

## 🚀 How to Use

### Admin Panel
1. Double-click footer  
2. Use options:
   - View → open console (F12)  
   - Export → download JSON  
   - Clear → reset data  

---

### Events
- Events load automatically from JSON  
- Use search bar to filter  
- Click "Register" to participate  

---

### Forms
- Fill all required fields  
- Enter valid email  
- Data saves automatically in browser
